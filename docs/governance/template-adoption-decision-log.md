# Template Adoption Decision Log

Use this file when applying the template to an existing project. Record how each
template rule or artifact maps to the real repository.

## Decision States

- `adopt`: use the template rule as written
- `adapt`: keep the intent, change the implementation for this repository
- `defer`: track as future work before enforcing
- `reject`: intentionally not applicable to this repository
