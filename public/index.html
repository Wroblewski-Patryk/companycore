How to connect Google Drive APIs
  1. In Google Cloud, enable Google Drive API, Google Docs API, and Google Sheets API for the project.
  2. Configure the OAuth consent screen. During testing, add the exact Google account you will use here as a Test user; otherwise Google can block sign-in before CompanyCore receives a code.
  3. Create an OAuth Web application client and add the Redirect URI below exactly as an authorized redirect URI.
  4. Paste the OAuth client ID and client secret below, save the OAuth client, create the OAuth URL, then open Google consent.
  5. If Google shows an unverified-app warning, use Advanced / continue only for your own test app, or finish Google's OAuth verification before wider use.
  6. After the authorization code is saved, load Drive folders, select the folders to import, save selection, then run import.
